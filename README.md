# Cynthia Ugwu Portfolio Clone

This is a front-end clone of Cynthia Ugwu’s portfolio website, created for learning and practice purposes.

## 🚀 Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- GSAP  
- Locomotive Scroll  

## ✨ Features
- Smooth scrolling experience  
- GSAP-powered animations  
- Custom cursor interaction  
- Loader animation  
- Responsive layout  

## 🎯 Purpose
The main goal of this project was to improve my understanding of modern front-end animations, smooth scrolling techniques, and interactive UI design.

This project is a clone created strictly for educational purposes. All design credits go to Cynthia Ugwu.
